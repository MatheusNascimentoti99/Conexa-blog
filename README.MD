## Blog Conexa ###

### Instalação ###
* `git clone https://github.com/MatheusNascimentoti99/Conexa-blog.git ` 
* `cd Conexa-blog`
* `cp .env.example .env`    
* `docker compose up`

### Acesso ###
* `http://localhost:80/ConexaBlog/`
